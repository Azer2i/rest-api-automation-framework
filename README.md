This project demonstrates API Test Automation using:

- Java
- RestAssured
- TestNG
- Data-Driven Testing (Excel)

## Features

- DataProvider with Excel integration
- Reusable utility classes
- Clean test structure

## Project Structure

- payload → request models
- endpoints → API calls
- utilities → reusable methods (Excel reader, config)
- tests → test cases

## Example

Data-Driven test runs the same API test with multiple data sets from Excel.
